# ESTAT API

ESTAT API is a fork of Track and Trace API (v1). It uses a separate instance of the TNT smart contract.

It should only be deployed on Test and Pilot environments, and nowhere else.

## Service configuration

Create a `.env.default.local` file locally. You can duplicate the content of `.env.default` or only set the necessary variables.
